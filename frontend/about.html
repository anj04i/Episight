About The Project

AI-Powered
Epidemiology
at Scale

EpiSight AI leverages Google Community Mobility Reports and distributed machine learning (PySpark + MLlib) to train a Gradient Boosting model that forecasts disease case counts from human movement patterns — making outbreak intelligence accessible through a simple REST API.

Technology Stack
  • PySpark
  • MLlib
  • Scikit-learn
  • Flask
  • JavaScript
  • HTML5
  • CSS3
  • Python 3
Model Performance
Algorithm Gradient Boosting Regressor
R² Score ≈ 0.60
Training Framework PySpark MLlib
Inference Layer Scikit-learn (Flask)
Input Features 4 Mobility Signals
Output Predicted Case Count
Data Sources

Training data is sourced from Google Community Mobility Reports — anonymised, aggregated datasets that capture mobility trends across categories relative to a pre-pandemic baseline.
